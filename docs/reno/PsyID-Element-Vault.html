The five axes v1.1
The measurement spine shared by both products. Each axis is orthogonal, scored 0–100, and points toward its plus pole (the reference direction a score of 100 indicates). Axis 5 — Emotional Response — is new in v1.1 and absorbs the composure/stability content that used to contaminate axes 3 and 4.

02 / The Code
Band scale new
The intensity rubric: a continuous 0–100 score bands into six reading levels per pole. Bands communicate; they never compute — every reliability and stability calculation runs on the raw score, never the band.

03 / Data Elements
Radar — pick one 3 options
Five axes need a pentagon, not a diamond. Three ways to draw it — same data, same data-values="0.32,0.71,0.54,0.68,0.38", switch with data-variant. Tell me which you prefer and I'll standardise on it across both documents. Values map each axis toward its plus pole (O·C·L·D·S).
Option A — Pentagon web · data-variant="pentagon"
Option B — Petal bloom · data-variant="petal"
Option C — Spoke gauge · data-variant="spoke"
Any variant also renders on dark / gradient · data-theme="gradient"
03 / Data Elements
Spectrum bar
One bipolar axis. data-value="0.32" (0 = minus pole, 1 = plus pole). Optional data-ci="0.24,0.42" shades the confidence interval around the thumb — the honest "likely range," not a false-precision point.
03 / Data Elements
Confidence interval bar new
The raw 0–100 score shown with its likely range — data-value="71" data-lo="64" data-hi="78". This is the visual that protects the re-test promise: next year's 66 shown inside the band reads as stability, not "your child changed."
03 / Data Elements
Axis pill
Compact bar for lists and passport summaries. Set data-label, data-value="0.71" and data-code="A4". Colour per axis via --ax.

The derived shadow — the least-developed axis, the opposite of the dominant preferences. It is a Level 2/3 interpretation, never a measurement: no code slot, no confidence band. Two life-stage framings of the same construct.
